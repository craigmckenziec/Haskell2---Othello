getCoord #

Arguments

:: String

String that is to be turned into a coordinate

-> (Int, Int)

Pair of ints that either represent a 2-d coordinate or a failure to turn into coordinates resulting in (-1,0)

Given an input string, convert it to a board coordinate as a pair of Ints e.g. getCoord D4 => (3,3) since coordinates are 0-based internally. or getCoord F2 => (5,1)
